# Coding-Quiz
The provided code represents a simple web application for a quiz game with a timer and the ability to save high scores. When you get an answer correct your score increments by 1 and at the end of the quiz yous ee the score. If you get an answer wrong your time decreases by 10 seconds and you dont recieve a point.
